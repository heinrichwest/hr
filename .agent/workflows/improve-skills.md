# Skill Improvement Process

Help improve the description and content of skills located in `.claude/skills/`.

1. **Confirm**: Ask which skills to improve.
2. **Analyze**: Read `SKILL.md` and related standards in `agent-os/standards/`.
3. **Rewrite**: Improve the `description` in frontmatter for better discoverability.
4. **Enhance**: Add a "When to use this skill" section.
5. **Advise**: Provide recommendations for further improvement.
